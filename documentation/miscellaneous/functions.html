ExponentiallyBackoff
ExponentiallyBackoff(maxRetryAttempts: number, initialDelay: number, maxDelay, disableAndUseConstantDelayOf?: number)

Function to implement exponential backoff in observables.

Parameters :
Name Type Optional Default value Description
maxRetryAttempts number No 3
  • The maximum number of retry attempts. Default is 3.
initialDelay number No 1000
  • The initial delay in milliseconds before the first retry. Default is 1000.
maxDelay No 2 * 60 * 1000
  • The maximum delay in milliseconds between retries. Default is 2 minutes.
disableAndUseConstantDelayOf number Yes
  • If provided, disables exponential backoff and uses this constant delay instead.
Returns : Observable<T>
  • A function that takes an Observable and returns an Observable with retry logic applied.

TimeoutBackoff
TimeoutBackoff(retryCallback, errorCallback?, maxRetries: number, initialDelay: number, maxDelay)

Creates a function that will call a retry callback with an exponentially increasing delay between each call. If the maximum number of retries is exceeded, an optional error callback is called. The delay between retries can be reset by calling the returned function with true.

Parameters :
Name Type Optional Default value Description
retryCallback No
  • The callback to be called on each retry.
errorCallback Yes
  • The callback to be called when the maximum number of retries is exceeded.
maxRetries number No 3
  • The maximum number of retries before the error callback is called.
initialDelay number No 1000
  • The initial delay in milliseconds before the first retry.
maxDelay No 2 * 60 * 1000
  • The maximum delay in milliseconds between retries.
Returns : void
  • A function that when called, will initiate a retry or reset the retry count.
